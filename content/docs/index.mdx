## What is Volts?

Volts is an AI-native SaaS boilerplate for indie hackers and small teams. It gives you everything you need to launch a production SaaS product with built-in AI capabilities.

## Tech Stack

- **Framework:** Next.js 16 with App Router
- **Language:** TypeScript (strict mode)
- **Database:** PostgreSQL + Drizzle ORM
- **Auth:** Auth.js (NextAuth v5) — GitHub, Google, email/password
- **Billing:** Stripe subscriptions + usage-based metering
- **AI:** Vercel AI SDK — OpenAI + Anthropic
- **Styling:** Tailwind CSS 4 + shadcn/ui
- **Deployment:** Vercel, Railway, Docker

## Features

- Multi-provider OAuth + credentials authentication
- Multi-tenant organizations with roles and invitations
- Stripe billing with subscriptions and customer portal
- Dual LLM provider support with streaming and tool calling
- AI usage tracking and rate limiting
- Admin dashboard with audit logs
- Landing page with pricing, FAQ, and waitlist
- Dark mode support
