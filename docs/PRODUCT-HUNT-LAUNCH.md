## FAQ — Prepared Responses

**Q: How is this different from other Next.js boilerplates?**
> Volts is AI-native. Most boilerplates give you auth and billing — we also include LLM integration patterns with OpenAI and Anthropic, streaming, tool calling, usage tracking, and rate limiting. If you're building an AI product, this saves you weeks.

**Q: Can I use this for multiple projects?**
> Yes. Buy once, use on every project you build. No per-project or per-seat licensing.

**Q: What tech stack does it use?**
> Next.js 16 (App Router), TypeScript (strict mode), PostgreSQL + Drizzle ORM, Auth.js v5, Stripe, Vercel AI SDK, Tailwind CSS 4 + shadcn/ui. Deploys to Vercel, Railway, or self-hosted via Docker.

**Q: Do I need to use both OpenAI and Anthropic?**
> No. The AI integration is modular — use one provider or both. Swap them out or add others. The patterns (streaming, tool calling, rate limiting) work with any provider supported by Vercel AI SDK.

**Q: Is this open source?**
> The codebase is yours to modify however you want. It's a commercial product with a one-time purchase, but there's no obfuscation or lock-in.

**Q: What kind of support is included?**
> Starter: community support. Pro: priority Discord support for 12 months. Team: priority support + a 1:1 onboarding session.

**Q: Can I self-host this?**
> Yes. Docker deployment config is included. Works with Coolify, Railway, or any Docker-compatible host. No vendor lock-in.

**Q: What if I need features that aren't included?**
> Volts is a starter kit — you own the code and can extend it however you want. The architecture is modular, so adding features doesn't mean fighting the framework.
